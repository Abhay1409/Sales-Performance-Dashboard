# Sales Performance Analysis Dashboard

## Project Overview
End-to-end sales analytics pipeline processing 52,000+ transactions 
across 3 years, 5 product categories, and 4 regions.

## Tech Stack
- Python (Pandas, NumPy)
- Power BI Desktop
- Power Query (M Language)
- DAX

## Key Highlights
- 52,000+ transactions processed
- ₹113 Cr total revenue tracked
- 2,500+ missing values resolved
- 4-page interactive dashboard
- 8+ KPIs tracked

## Project Structure
- 01_generate_data.py — Dataset generation
- 02_clean_data.py — Data cleaning pipeline
- SalesPerformance.pbix — Power BI dashboard
- SOP.md — Maintenance documentation

## Dashboard Pages
1. Overview — Revenue, Orders, KPI cards
2. Profitability — Margins, Underperformers
3. Regional Analysis — Region & Salesperson
4. Product Analysis — Deep dive by product
